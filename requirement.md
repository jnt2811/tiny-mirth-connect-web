# Yêu cầu ứng dụng

Tao muốn mày làm 1 web react typescipt tanstack router, tanstack query, rsbuild, mantine v8 (table thì dùng mantine-datatable) có tính năng đăng nhập, tài khoản mirth-connect là admin/admin, đọc cái mirthapi.json xem có api đăng nhập không, nếu có thì tao nghĩ là sẽ dùng nó để xác thực chỗ đăng nhập của web mình luôn. Đăng nhập thành công thì mới cho vào dashboard, dashboard sẽ có 1 cái header và content bên dưới. Trên header sẽ có 2 menu item là Bảng điều khiển (dashboard) và Kênh (channel), tính ra là 2 page đó. tanstack router thì dùng routetree gen. Page dashboard sẽ bao gồm inbound message, outbound message, protocol của kênh. Page channel sẽ bao gồm các cấu hình dưới dạng readonly để đảm bảo stable. Lấy dữ liệu và hiển thị như thế nào thì mày cứ lấy ra rồi hiển thị flexible giữa lựa chọn dạng bảng hay dạng danh sách gì đó.
